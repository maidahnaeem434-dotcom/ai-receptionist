# AI Receptionist

## Overview

An AI-powered voice receptionist designed to handle inbound customer calls, answer common questions, qualify leads, and schedule appointments automatically.

## Features

- Answer inbound calls
- Qualify leads
- Book appointments
- Handle FAQs
- Transfer qualified leads
- CRM integration

## Tech Stack

- VAPI
- n8n
- OpenAI
- Google Calendar
- Twilio

## Workflow

1. Customer calls business
2. AI answers the call
3. Collects customer information
4. Answers questions
5. Books appointment or transfers lead
6. Updates CRM

## Use Cases

- Real Estate
- Healthcare Clinics
- Home Services
- Agencies
- Local Businesses
